/overview

Página experimental. Mismos widgets autodescubiertos, misma API (/api/v1/overview/widget/<id>) y mismo diseño que /overview (reutiliza web_admin.css + la paleta de acentos y el markup de tarjeta de producción); frontend con Alpine.js (local, sin build). El layout se guarda en tu cuenta vía PUT /api/v1/users/me/preferences (el mismo dashboard_layout que /overview). Las tablas reutilizan la tarjeta de producción pero con un renderer de celdas genérico (las celdas muy específicas se simplifican).
Nota de seguridad: esta página sirve una CSP propia con 'unsafe-eval' (solo aquí) porque Alpine evalúa sus expresiones con new Function(), que la CSP global prohíbe.
